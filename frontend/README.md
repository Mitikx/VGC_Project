# VGC-Pro Frontend

## 🚀 Étape 6 — Nouvelles features

### Mise à jour

```bash
cd frontend
npm install
npm run dev
```

### Nouvelles pages

- `/quick` — **Saisie rapide** (30 sec, résultat + 4 Pokémon)
- `/games/:id` — **Détail d'une partie** (consultation + édition + partage)
- `/profile` — **Mon profil** (bio + activation profil public)
- `/u/:username` — **Profil public** d'un autre user
- `/share/:token` — **Partie partagée** (consultable sans compte)

### Nouvelles fonctionnalités

- **Calendrier mensuel** dans la page Parties
- **Édition** d'une partie depuis sa page de détail
- **Partage** d'une partie via lien public
- **Profils publics** avec stats agrégées
